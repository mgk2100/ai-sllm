# sl-sw-sllm 리스크 분석

## 리스크 목록

### 1) 재현성 부족
- 심각도: High
- 근거: README가 매우 짧고 실행 절차가 스크립트에 분산됨
- 단기 완화: 표준 실험 절차(데이터 준비-학습-검증) 문서화
- 중기 개선: 실험 설정 버전관리(Hydra config + 결과 메타데이터) 강화

### 2) 환경 종속 경로 하드코딩
- 심각도: Medium
- 근거: 실행 스크립트에 `data/...` 경로와 고정 인자 사용
- 단기 완화: 스크립트 인자화 및 기본값 문서화
- 중기 개선: 공통 설정 파일로 경로/리소스 정책 통합

### 3) 운영 전환 준비도 부족
- 심각도: Medium
- 근거: 학습/서빙 구성은 존재하나 배포/모니터링 가이드 부재
- 단기 완화: 모델 산출물 규격 및 검증 기준 정의
- 중기 개선: 추론 성능/품질 KPI와 배포 체크리스트 수립
