# sl-sw-sllm 초기 분석 요약 (2026-02-13)

## 프로젝트 목적
`sl-sw-sllm`은 sLLM 데이터 전처리/학습 실험을 위한 저장소다.
`preprocess` 스크립트와 `unsloth` 학습 코드, `vLLM` 예시 구성이 공존하며,
주요 사용 맥락은 데이터셋 생성과 Continual Pretraining(CPT) 흐름이다.

## 현재 상태
- 성숙도: 실험/연구 중심 레포지토리
- 실행 방식: Python 스크립트 + 셸 스크립트 기반 수동 실행
- 문서화 수준: README가 매우 간략하여 실행 지식이 스크립트 내부에 있음

## 핵심 이슈 Top 3
1. 실행 절차/데이터 전제조건 문서가 부족해 재현성 리스크가 크다.
2. 스크립트의 하드코딩 경로(`data/...`) 의존으로 환경 이식성이 낮다.
3. 학습/추론 관련 샘플 설정은 있으나 표준 운영 플로우가 명확하지 않다.

## 다음 액션 Top 3
1. 데이터 준비-학습-평가 단계를 문서화한 실행 가이드를 추가한다.
2. 경로/모델/엔드포인트를 환경변수 또는 설정 파일로 분리한다.
3. 최소 재현 실험(MRE) 절차와 결과 검증 체크리스트를 정의한다.
